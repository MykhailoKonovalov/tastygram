# Tastygram

#### My pet project for studying Symfony API Platform, React JS and other stuff that I decide to use in the process.

#### The topic of the project is a culinary blog

### Setup instructions:
1. Install docker and docker-compose.
2. ```cp api/.env api/.env.local```
3. ```make run```
